# 🗺️ Descarga el mapa de México aquí

Este archivo `mexico.json` NO está en el ZIP por tamaño/licencia. Descárgalo:

## Opción 1: terminal (recomendado)

```bash
curl -L https://raw.githubusercontent.com/PhantomInsights/mexico-geojson/main/2022/mexico.json -o mexico.json
```

## Opción 2: navegador

1. Abre esta URL: https://raw.githubusercontent.com/PhantomInsights/mexico-geojson/main/2022/mexico.json
2. Click derecho → "Guardar como" → `mexico.json`
3. Guárdalo en esta carpeta (`docs/data/`)

Una vez descargado, recarga el dashboard y el mapa aparecerá automáticamente.

## ¿Qué pasa si no lo descargas?

El dashboard funciona igual, pero en la sección del mapa verás un mensaje
indicando que el archivo no fue encontrado. Las demás gráficas funcionan
normal sin esto.
